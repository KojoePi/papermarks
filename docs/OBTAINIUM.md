# Obtainium setup

Papermarks is intended for GitHub Release installation.

Recommended release asset name:

```text
papermarks-vX.Y.Z.apk
```

Recommended repository setup:

```text
https://github.com/YOUR_USER/papermarks
```

In Obtainium, add the repository as a GitHub source and track releases. Obtainium will check GitHub Releases for newer APK assets.
